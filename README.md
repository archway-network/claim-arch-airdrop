# Claim Archway Airdrop demo

## Install dependencies

- [Deno](https://deno.land/)

## Setup

Add your mnemonic to the `.env` file. You can use the `.env.example` as a template.

## Run

TO claim the `connect` step:

```bash
deno task start
```
